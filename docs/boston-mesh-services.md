# Boston Mesh Services

This page lists the core websites and services run by **Greater Boston Mesh**.

## MeshCore Live Map

Live MeshCore traffic map that renders nodes, routes, and activity in real time on a Leaflet map.
Made by local resident [Yellowcooln](https://github.com/yellowcooln).

- https://live.bostonme.sh/

## MeshCore Health Check

MeshCore observer coverage app that generates test codes, watches MQTT observer receipts, and scores real- world message reachability with per-observer path details. 

- https://healthcheck.bostonme.sh/

## Meshtastic Malla Map

Malla is a web analyzer for Meshtastic networks based on MQTT data.

- Malla host: https://malla.bostonme.sh/map

## MQTT (MeshCore)

- MQTT broker: `mqttmc01.bostonme.sh` (WebSockets/TLS, port `443`)
- MeshCore MQTT setup: https://bostonme.sh/docs/MeshCore/meshcore-mqtt

## MeshCore MQTT Dashboard

Live MQTT dashboard for node presence, roles, traffic stats, and broker telemetry.
Made by local resident [Yellowcooln](https://github.com/yellowcooln).

- https://mcmqttdashboard.bostonme.sh/

## MQTT (Meshtastic)

- MQTT broker: `mqttmt01.bostonme.sh` (MQTT, port `1883`)
- Meshtastic MQTT setup: https://bostonme.sh/docs/Meshtastic/meshtastic-mqtt

## Notes

- MeshMapper and LetsMesh links are used by the community but are external services.
