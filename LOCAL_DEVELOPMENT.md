# Local Development

This website is built with [Docusaurus](https://docusaurus.io/).

## Install Dependencies

Using yarn:

```bash
yarn
```

Using npm:

```bash
npm i
```

## Run Locally

Using yarn:

```bash
yarn start
```

Using npm:

```bash
npm run start
```

This starts the local development server at `http://localhost:3000` with hot reload.
